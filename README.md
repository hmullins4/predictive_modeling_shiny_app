# Statistical Modeling, Time Series Prediction, and Interactive Shiny App Development in R

This multi-part project applied R to statistical modeling, predictive analytics, and interactive application development across several datasets and use cases. I analyzed the World Happiness dataset using descriptive statistics, linear and polynomial regression, LASSO, SVM, correlation analysis, and mutual information to study relationships between happiness and its contributing factors. I also built a lagged time-series forecasting workflow for cryptocurrency prices using HDF5 storage, train-test splits, linear regression, LASSO, and SVM to compare predictive performance. In addition, I developed a Shiny web application with Leaflet and geospatial distance calculations to display Florida locations, geocode user input, and compute distance from UCF. Overall, the project demonstrates skills in regression modeling, feature selection, time-series analysis, geospatial visualization, and interactive R application design.
